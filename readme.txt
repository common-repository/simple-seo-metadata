Edit meta description, meta keywords and title for each page, post, post type.

== Description ==
Add meta box to posts and custom posts containing 3 fields

1. SEO title
1. Meta keywords
1. Meta description

If SEO title not empty it will override the default page title.

* easy to use
* easy to install
* works with all post types

= Multilingual & RTL supported. =
We've integrated RTL and multi language support, as well as RTL typography support.

== Installation ==

= Minimum Requirements =

* WordPress 5.0 or greater
* PHP version 5.6 or greater
* MySQL version 5.0 or greater

= We recommend your host supports: =

* PHP version 7.0 or greater
* MySQL version 5.6 or greater

== Installation ==
1. Upload "simple-seo-meta" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Edit the fields inside the post or page.
